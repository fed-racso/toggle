Tab Icons

Displaying icons inside the tabs is possible with a little extra CSS. See the docs for alternative ways to add tab icons.

Initialization:

$(document).ready(function() {
    $('#tab-icons').slidetabs();
});

CSS for the tab icons:

div#tab-icons div.st_tabs a { 
	position: relative; 
	text-indent: 50px; 
}
div#tab-icons div.st_tabs span.st_icon { 
	position: absolute; 
	top: 14px; 
	left: 15px; 
	display: block; 
	width: 20px; 
	height: 20px;
	background: url("../img/tab-icon.jpg") no-repeat 0 -20px transparent;
}
div#tab-icons div.st_tabs a.st_tab_active span.st_icon,
div#tab-icons div.st_tabs a:hover span.st_icon {
	background-position: 0 0;
}
